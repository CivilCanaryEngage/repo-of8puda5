# A股板块主力行为计算工具

每天收盘后自动拉取东方财富板块资金流数据，计算各板块的主力行为并生成报告。

## 公式

- 主力暗盘 = 主力资金 - 散户资金
- 主力强度 = 主力暗盘 / 成交额 × 100
- 主力行为：多日复合判别（资金方向 × 价格表现 × 散户方向，接近主流量化工具做法）：

| 主力行为 | 判别条件 | 含义 |
| --- | --- | --- |
| 抢筹 | 当日强度 ≥ 3 且 涨幅 > 0 | 主力强力流入且放量上攻 |
| 出货 | 当日强度 ≤ -1 且（散户净流入 或 5日主力占比 ≤ -1）| 主力撤离、散户接盘或多日持续流出 |
| 建仓 | 5日主力净占比 ≥ 1 且 当日强度 > -1 | 多日持续吸筹，当日未明显流出 |
| 洗盘 | \|当日强度\| < 1 且 涨幅 ≤ 0 且 散户净流出 | 资金平衡、缩量回调、散户离场 |

以上规则自上而下优先匹配；均不命中时按单日强度兜底（≥3 抢筹 / [1,3) 建仓 / (-1,1) 洗盘 / ≤-1 出货）。

## 数据说明

数据来源为东方财富公开的板块资金流接口（无需 key，仅用标准库，无第三方依赖）：

- 主力资金 = 超大单 + 大单净流入
- 散户资金 = 小单净流入（中单视为中户/游资，不计入散户；东方财富资金流为零和数据，若把中单+小单都算散户，会导致散户恒等于 -主力、暗盘恒等于 2×主力，指标失真）
- 成交额、涨幅取当日板块实时/收盘数据
- 5日涨幅、5日主力净占比取自同一接口的多日字段；历史明细中的 5日占比由滚动 5 日窗口计算

## 使用

```bash
# 行业板块（默认），输出全部并生成 CSV/HTML 报告到 output/
python3 main_force.py

# 概念板块，终端只显示前 30 条
python3 main_force.py --kind concept --top 30

# 地域板块
python3 main_force.py --kind region
```

输出字段：板块、涨幅、5日涨幅、成交额、主力资金、散户资金、主力暗盘、主力强度、5日主力占比、主力行为。
报告保存为 `output/日期_类型.csv` 与 `output/日期_类型.html`。

## 桌面客户端 (Windows)

`gui.py` 提供图形界面（Tkinter，标准库自带）：板块类型切换、行为筛选、点击表头排序、手动刷新、每日 15:30 自动刷新、导出 CSV。
表格最前面固定展示上证指数、创业板指、科创50三个大盘指数，其余板块按市场热度（成交额）降序排列。
双击任意行（含置顶指数）可查看每日主力行为明细，历史窗口可切换近 7 天 / 近 14 天，并统计期间累计主力净流入与累计暗盘；顶部搜索框支持个股/板块（代码、中文名、拼音缩写），回车直达，多个匹配时弹出选择列表；支持 F5 刷新、斑马纹表格与表头排序箭头指示。

其他功能：

- **主力异动榜**：一键筛选"前期多日持续吸筹（5日主力占比≥1）且当日放量抢筹"的板块——典型启动信号，按当日强度降序，双击查看历史。
- **行为变化提醒**：刷新后行为切换为"抢筹"的板块以红底高亮，并在状态栏提示。
- **限流优化**：历史接口（push2his）对频繁请求限流；工具带递增退避重试与 10 分钟本地缓存，同一标的短时间重复查看不再请求接口。若仍被限流会降级为仅当日数据并在历史窗口提示"⚠ 接口限流"，稍后重新打开即可取到完整历史。

```bash
python3 gui.py
```

Windows exe 由 GitHub Actions 自动打包（PyInstaller）：在仓库 Actions 页面的 "Build Windows Client" 工作流中下载 `A股主力行为-windows` 构件，解压后双击 exe 即可运行，无需安装 Python。

## 每日收盘后自动运行

A股收盘为北京时间 15:00，建议 15:30 运行。crontab 示例（服务器为北京时间）：

```cron
30 15 * * 1-5 cd /path/to/astock-main-force && python3 main_force.py >> run.log 2>&1
```

若服务器为 UTC 时间（北京时间 = UTC+8，15:30 CST = 07:30 UTC）：

```cron
30 7 * * 1-5 cd /path/to/astock-main-force && python3 main_force.py >> run.log 2>&1
```
